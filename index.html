Newtonův gravitační zákon

Měň hmotnosti m₁, m₂ a vzdálenost r. Vzorec i výsledek se přepočítají automaticky.
F = 0 N
G = 6,67430×10⁻¹¹ N·m²·kg⁻²
Vzorec:
F = G · (m₁ · m₂) / r²
Dosazení:
F = (6,67430×10⁻¹¹) · (1 · 1) / (1)²
F = 0 N
Nastavení
Vysvětlení (statika / síla)
Gravitační síla působí vždy přitažlivě mezi dvěma hmotnými tělesy. Čím jsou tělesa těžší, tím je síla větší. Čím jsou dál od sebe, tím síla rychle klesá (je nepřímo úměrná r²).
Popis veličin
F … gravitační síla (N)
G … gravitační konstanta (6,67430×10⁻¹¹ N·m²·kg⁻²)
m₁, m₂ … hmotnosti těles (kg)
r … vzdálenost středů těles (m)
Tip pro žáky
Když se r zdvojnásobí, síla se zmenší na čtvrtinu. (Protože r².)
